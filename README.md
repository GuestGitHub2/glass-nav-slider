## Usage

Clone and open `index.html` in your browser, or deploy to GitHub Pages.

### Available Controls

- **Position X/Y** - Adjust slider position
- **Opacity** - Material transparency
- **Blur** - Backdrop blur strength
- **Saturation** - Color saturation level
- **Glassy** - Glass effect brightness (0-100%)
- **Overshoot** - Animation bounce control (0-100%)
- **Border/Shadow** - Edge effects
